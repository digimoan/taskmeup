Features

Create Projects & Tasks
Our projects work like a folder — you can name them, assign team members, and create tasks. Members can log the start and finish of their tasks.

Overview Tasks
On “My Tasks” each team member can overview all their assigned tasks on the project. They’re organized by colors as “To do”, “Doing” and “Done”.

Overview Time Spent
At the end of each project, you can use the “Time Tracker” tab to see how long each team member spent on each task and other sorts of time metrics.
